# E-Commerce Data Analysis

## Overview
SQL analysis of e-commerce sales data to uncover business insights.

## Key Findings
- Fashion category has the highest return rate (12.17%)
- Discount percentage has no significant impact on customer ratings
- Payment methods are evenly distributed

## Tools Used
- SQL (SQLite)
- DB Browser for SQLite

## Files
- ecommerce_analysis.sql — SQL queries for all analysis
